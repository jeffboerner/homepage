Below is the safe configuration space of 3 robots in the interval [0,1]. The planes, press 'p' to see them, represent the diagonal that must be removed for the configuation space to be safe. It is safe because no robots can occupy the same place at the same time.

(Click on image to interact with the application. WebGL must be enabled.)



Controls:
p: place diagonal planes
g: separate the connected components
c: color the connected components

Drag the cube to rotate it in 3 dimensions.
